## 1.Markdown이란?
- .md(Markdown) 확장자로 웹에서 약속된 기호, 패턴으로 글을 예쁘게 작성할 수 있게 도와주는 기술
- 텍스트로 쉽게 사용해서 쉽게 읽을 수 있도록 하는게 목적
